Boilerplate for Django
======================

Features
 * SCSS
 * CSS/JS Compression
 * Admin UI updated
 * Login Screen
 * Custom Auth User with email adress
 * FB login & registration

Installation:
 * pip install -r requirements.txt
 * bower install

Server
 * Django
 * Rest Framework

Client
 * Angular
 * Angular UI Router
 * Twitter Bootstrap
 * Bootstrap Material Design
 * Font Awesome
